# AI đang sở hữu những kỹ năng nào của một Mobile Developer, mức độ thành thạo ra sao? Khám phá và đánh giá bộ kỹ năng của AI trong phát triển ứng dụng di động theo phương pháp Vibe Coding

### Câu hỏi nghiên cứu
1.AI sở hữu những nhóm kỹ năng nào trong Mobile Development?

2.Mức độ hiệu quả của từng kỹ năng?

3.Kỹ năng nào AI thực hiện tốt hơn con người?

4.Kỹ năng nào vẫn cần sự tham gia của lập trình viên?

5.Mô hình Human-AI Collaboration nào phù hợp cho Mobile Development?

### Mục tiêu

Nghiên cứu khả năng của các mô hình AI khi tham gia vào quy trình phát triển ứng dụng di động bằng Flutter dưới vai trò một Mobile Developer ảo. Thông qua các bài thực nghiệm thực tế, đề tài hướng đến việc xác định các nhóm kỹ năng AI đang sở hữu, đánh giá mức độ hiệu quả của từng kỹ năng, đồng thời đề xuất mô hình cộng tác giữa con người và AI trong phát triển ứng dụng di động.

### Nội dung nghiên cứu

Các nhóm kỹ năng được khảo sát bao gồm:

* Code Generation
* Debugging
* Refactoring
* UI Generation
* UI-to-Code
* UX Suggestion
* Architecture Design
* State Management Design
* Database Design
* API Integration
* Testing
* Performance Optimization
* Security Analysis
* Requirement Analysis
* Documentation Understanding
* Product Thinking

* Đánh giá khả năng đề xuất cải thiện UX.

#### Kết quả đầu ra
